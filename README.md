# Dashboard de Gestión de PQRS

## Descripción
Dashboard desarrollado en Power BI para el seguimiento y control de PQRS en contratos de mantenimiento.

---

## Objetivo
Analizar tiempos de gestión, cumplimiento de SLA y desempeño de contratistas.

---

## Herramientas
- Power BI  
- SQL (Oracle)  
- DAX  
- Power Query  

---

## KPIs
- Total PQRS  
- PQRS activos  
- Tiempo promedio de gestión  
- Cumplimiento SLA  

---

## Impacto
- Identificación de retrasos en PQRS  
- Mejora en el control de contratistas  
- Soporte a decisiones operativas
